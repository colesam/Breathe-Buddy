Frameworks Used

AngularJS

AngularJS is a JavaScript framework that we used to get two way data-binding within our application. This can be seen in the location controls when the center of the map is changed.

We also used ng-includes to break our site into two html sections, one for the application and one for the project about page. By using these ng-includes we are able to load these two different sections without refreshing the entire page. Test it out by switching between them!

JQuery

JQuery came bundled with Bootstrap so we decided to use it for DOM manipulation because its syntax is cleaner. If we choose to add animations or make the navbar/site responsive JQuery is already loaded and ready to make those additions easier.

Bootstrap

We used Bootstrap for it's grid system. The grid system and Bootstrap's flexbox classes allowed us to set up the user interface for our application without writing too much custom CSS.
